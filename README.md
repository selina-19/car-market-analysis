# Car Market Analysis

## Overview
This project explores a large vehicle dataset (~400,000 records) to analyse key factors affecting car prices, including mileage, fuel type, vehicle condition, and brand characteristics.

## Objective
To identify patterns, relationships, and trends in the automotive market that influence pricing and consumer preferences.

## Key Analysis Areas
- Feature understanding and distribution analysis  
- Data cleaning and preprocessing  
- Feature engineering (e.g., mileage categories, vehicle age)  
- Market segmentation and subsetting  
- Analysis of relationships between variables  

## Key Insights
- Vehicle price decreases with increasing mileage (negative correlation)  
- New and low-mileage vehicles command higher prices  
- Petrol and diesel dominate the market, while hybrid/electric vehicles show emerging trends  
- Brand and body type combinations significantly influence pricing  
- SUVs and hatchbacks dominate the market distribution  

## Tools Used
- Python  
- Pandas, NumPy  
- Matplotlib, Seaborn  

## Dataset
~400,000 vehicle records including features such as mileage, fuel type, vehicle condition, and registration year.
